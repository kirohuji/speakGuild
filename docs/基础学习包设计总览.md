## 二、设计理念

### 2.1 核心原则：场景优先，语法穿插

不按传统教材的"语法点 → 例句"模式，而是**按真实使用场景组织**。学习者先进入场景（自我介绍、问路、点餐、打电话），在场景中自然遇到语法点。

## 三、10 个包逐一说明

### 基础① · be动词与陈述句基础（L1）

- **核心**：I am / You are / He/She/It is / We/They are
- **场景**：自我介绍 → 描述事物 → 描述他人 → 日常寒暄
- **设计思路**：英语每个句子必须有动词，be 动词是地基。从"我是谁"到"打招呼"，逐步扩展场景。
- **特殊附件**：be 动词速查卡（肯定/否定/疑问三列对照）

### 基础② · 一般动词与日常动作（L1）

- **核心**：I eat / I like / I want / I need / I have + 否定(don't) + 疑问(Do you?)
- **场景**：日常习惯 → 喜好需求 → 日程安排 → 购物出行 → 餐饮社交
- **设计思路**：从 be 动词（描述状态）升级到一般动词（描述动作），覆盖日常生活的完整链条。
- **规模**：10 话题 × 68 句块（所有包中最大），因为日常动作是最常用的交流内容。

### 基础③ · 第三人称与一般现在时（L1）

- **核心**：He eats / She likes / doesn't / Does he...? + 所有格 his/her
- **场景**：描述家人 → 他/她的日常 → 对比你我 → 职业与叙事
- **设计思路**：中文动词不变形，英语第三人称必须加 -s——中国学习者最顽固的错误，必须单独一个包打透。加入了讲故事和画面描述的第三人称叙事练习。

### 基础④ · 疑问句实战（L1）★

- **核心**：Where/What/When/Who/Why/How + be/do/does 问句
- **场景**：出行与交通 → 消费与服务 → 社交与寒暄 → 电话与联络 → 应急与求助
- **设计思路**：**最重要的口语能力包**。不按语法分类（Yes/No、Wh-、How），按"你在什么场景下需要问什么"组织。严格只用 ①-③ 已学语法（be/do/does），不使用情态动词。
- **特殊附件**：疑问词速查卡（14 种问句类型 × 脑内弹出词映射）
- **语法边界**：本包刻意避开 can/should/will/did——这些情态和过去时间句在 ⑦ 才系统练习。

### 基础⑤ · 现在进行时 vs 一般现在时（L1~L2）

- **核心**：I am eating vs I eat / 频率副词 / have 的双面性 / get 的日常用法
- **场景**：此刻正在做 → 临时 vs 习惯 → 频率副词 → have 与 get 时态对比
- **设计思路**：中文"正在"加一个字，英语用完全不同的时态。通过双列对比表（习惯 vs 此刻）建立时态意识。have（拥有 vs 吃/经历）和 get（习惯 vs 正在变得）是 L1 高频错误来源，专项突破。

### 基础⑥ · 一般过去时 & 完成时入门（L1）

- **核心**：was/were / 规则-ed / 30 个不规则动词（原→过→过分） / have been / for/since
- **场景**：过去的状态 → 过去的动作 → 讲述经历 → 完成时入门
- **设计思路**：过去时和完成时共享"过去分词"这个核心形式，放在一起学打通动词变形。完成时仅教 L1 级别（去过/做过/还没做/多久了），不做与过去时的细微辨析。
- **特殊附件**：30 个核心不规则动词三列速查卡（原形→过去式→过去分词）

### 基础⑦ · 将来时与情态动词（L1~L2）

- **核心**：will / be going to / can / must / should / may / have to
- **场景**：计划与预测 → 能力与请求 → 建议与义务 → 情态疑问句
- **设计思路**：将来时 + 情态动词共享"动词后面永远接原形"的规则。情态疑问句（Can I...? Could you...? Should I...?）是本包独有——基础④ 避开了它们，到这里才系统练习。
- **与④的衔接**：④ = be/do/does 问句，⑦ = can/could/should/would 问句，互补覆盖全部问句类型。

### 基础⑧ · 连词与从句入门（L1~L2）

- **核心**：and/or/but / because/so / when/while/before/after / if/unless / although / who/which/that
- **场景**：并列与转折 → 时间与条件 → 让步与定语
- **设计思路**：连接词是"语言胶水"，把简单句粘成段落。从 and/but 开始，逐步引入 because/if/when，最后到定语从句入门。

### 基础⑨ · 描述与比较（L1~L2）

- **核心**：比较级(-er/more) / 最高级(-est/most) / as...as / too/enough / there be / 介词
- **场景**：比较事物 → 描述位置 → 程度表达
- **设计思路**：描述力 = 交流力。从 big → bigger → the biggest，加上 there be 存在句和介词，让学习者能"画出画面"。

### 基础⑩ · 综合表达（L1~L2）

- **核心**：不定式 vs 动名词（to do vs doing） / would like / 礼貌请求 / for/since
- **场景**：喜好与活动 → 礼貌与请求 → 经历与持续
- **设计思路**：最后一包 = 升维。like swimming（动名词）vs want to go（不定式），从"课本英语"走向"真实英语"的桥梁。

## 七、整体评价

### 优势

1. **递进体系严谨**：be动词 → 一般动词 → 第三人称 → 疑问句 → 进行时 → 过去时 → 将来时/情态 → 连词 → 比较级 → 综合，逻辑链完整
2. **场景驱动设计**：④ 是典范——"你在街头怎么问路？在餐厅怎么点单？打电话怎么找人？"比"先学 Yes/No 问句再学 Wh- 问句"有效得多
3. **中国学习者痛点精准**：每个包都标注了母语干扰点（be 动词缺失、第三人称忘加 -s、疑问词后置等）
4. **句块设计扎实**：每个句子都是可直接背诵使用的完整对话单元，配有替换槽位练习
5. **语法边界清晰**：④ 严格避开未学语法，⑦ 才引入情态问句——学习者不会遇到"还没学就要用"的困境

### 待加强

1. **⑧ 和 ⑩ 有 L2 超纲内容**：定语从句和动名词/不定式辨析应降级或明确标注为 L2 预备
2. **⑧⑨⑩ 句块量偏少**（40 vs 54-70），可适当扩展
3. **基础知识点缺口**：数字、时间、星期/月份、冠词、祈使句、可数/不可数名词尚未系统覆盖
4. **ink 脚本和 CSV 数据均未根据最新 md 更新**（这是后续的数据生成工作）
