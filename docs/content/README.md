## 文件格式说明

每个单元 `.md` 包含：
- **剧情概述** — 这条链的故事背景
- **对话任务** × 2~4 — 每个任务的 NPC、情境、目标句块、对话流程
- **句块库** — 8~12 个核心表达块（IELTS 6~8 难度），各单元不重复
- **词汇库** — 12~18 个话题词汇（IELTS 6~8 难度），各单元不重复

## 设计原则

- **词汇不重复**：每个单元的词汇库围绕该单元主题独立设计，确保用户在不同单元学到的是新词
- **句块有侧重**：不同单元侧重不同交际功能（请求、说服、抱怨、安慰、解释…）
- **对话即练习**：每个任务都从开口开始，句块和词汇作为提示和复盘素材
