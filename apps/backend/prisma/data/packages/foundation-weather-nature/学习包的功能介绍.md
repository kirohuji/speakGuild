# 基础⑧ · 连词与从句入门 · 并列转折 · 因果让步 · 时间条件（L1~L2）

## 学习包概览

本学习包是 **Foundation 基础系列** 的 **第八包**。你将学习英语的 **连接词和基本从句**——把简单句连接成复杂句，表达因果、转折、时间、条件等逻辑关系。

> 💡 **为什么是第八包？** 有了前面的七包，你已经能说各种简单句了。但真正的交流需要连接想法——"我去了但是没找到""如果下雨我就不去了""她来之前我吃完了"。连词和从句让你从"说短句"升级到"说段落"。

### 本包核心语法

| 连接类型 | 连接词 | 示例 |
|:---------|:-------|:-----|
| **并列** | and / or / but | I like coffee and tea. / Is it big or small? |
| **因果** | because / so | I stayed home because I was sick. / I was tired so I went home. |
| **时间** | when / while / before / after | When I arrived, she was eating. / I always brush my teeth before bed. |
| **条件** | if / unless | If it rains, I'll stay home. / I won't go unless you come. |
| **让步** | although / even though | Although it rained, we had fun. |
| **定语从句** | who / which / that | The man who helped me was kind. / The book that I read was great. |
