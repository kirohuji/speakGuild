# 动词短语与固定搭配大全（L2~L3）

## 学习包概览

本学习包是 **语法专项系列** 中最**重要**的组成部分之一，聚焦英语 **80+ 个高频动词短语与固定搭配**。**L2~L3（中级基础~中级提高）** 定位，按 **介词小品词**（up/down/out/off/on/in/over/back）和 **核心动词搭配**（make/take/have/do/get/keep/set/come）分类，系统覆盖日常口语和写作中最实用的表达。

> 动词短语（phrasal verbs）是英语母语者日常交流的核心，**中国学习者最缺的就是这个**——课本学的是单个动词，老外说的全是动词短语。本包按小品词语义分组，每组大量句块反复操练，熟能生巧。

### 句块（chunk）vs 句型（pattern）

本包包含 **两类数据**，分工不同：

| | 句块（chunk） | 句型（pattern） |
|--|-------------|----------------|
| **是什么** | 可直接背诵的 **完整句子** | 带替换槽位的 **句型框架** |
| **作用** | 背下整句，脱口而出 | 替换小品词或宾语，举一反三 |
| **例子** | `I've run out of milk. I need to buy some.` | `I've run out of ___.`（slot: milk / time / money） |
| **对应文件** | `chunks.csv` | `sentence_patterns.csv` |

> **学习路径**：先背句块（语境中记短语）→ 再练句型（替换不同名词/小品词）→ 最终在对话中自然使用。

### 为什么动词短语这么重要？

| 场景 | 课本英语（不自然） | 地道英语（动词短语） |
|------|------------------|-------------------|
| 放弃 | I will not stop trying. | I will never **give up**. |
| 查明 | I need to discover the truth. | I need to **find out** the truth. |
| 拒绝 | He rejected the offer. | He **turned down** the offer. |
| 期待 | I anticipate your reply. | I **look forward** to your reply. |
| 容忍 | I cannot tolerate the noise. | I can't **put up with** the noise. |

> **目标**：让学习者从"课本英语"转向"地道英语"，口语和听力同时提升。

- **前置要求**：掌握基础动词和名词

### 学习目标

完成本包学习后，学习者能够：

1. **UP 短语**：用 give up / wake up / pick up / show up / grow up / cheer up / make up 等描述日常场景
2. **DOWN 短语**：用 calm down / break down / turn down / slow down / settle down 等表达状态变化
3. **OUT 短语**：用 find out / work out / figure out / hang out / point out 等表达发现、锻炼、社交
4. **OFF/ON 短语**：掌握 take off / turn off / put off / carry on / hang on / move on 等高频动作
5. **IN/OVER/BACK 短语**：用 give in / come over / call back / think over / drop in / pay back 等表达社交与决策
6. **MAKE/TAKE/HAVE/DO 搭配**：make a decision / take care of / have fun / do a favor 等最实用名词搭配
7. **GET/KEEP/SET/COME 搭配**：get ready / keep in touch / come true / set up / get along / come up with 等
8. **介词固定搭配**：interested in / good at / afraid of / depend on / proud of / used to 等
9. **地道转换**：将书面化的单个动词替换为更地道的动词短语（如 tolerate → put up with）

## 数据生成指引

### chunks.csv  vs  sentence_patterns.csv

| 文件 | 内容 | 示例 |
|------|------|------|
| **`chunks.csv`** | 完整句子（直接可背） | `I've run out of milk. I need to buy some.` |
| **`sentence_patterns.csv`** | 替换框架（举一反三） | `I've run out of ___.`（slot: milk / time / money） |

### 生成步骤

1. 先写 **`chunks.csv`**：从上表直接取句块，确保每个都有
2. 再从每个句块提取 **`sentence_patterns.csv`**：把核心短语挖空，变成替换框架
3. 确保 `episode_chunks.csv`、`scene_vocabulary.csv` 等关联数据同步更新
