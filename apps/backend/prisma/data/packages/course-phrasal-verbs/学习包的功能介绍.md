# 《漫语町》动词短语专项课程包

> 版本：v0.1  
> 用途：用于英语学习 App / 动词短语专项练习  
> 类型：course（课程型）
> 核心方向：按介词/副词小品词分类（up/down/out/off/on/in/over/back），系统性掌握英语核心动词短语。

---

## 课程模块

| 场景 | 核心短语 | 数量 |
|------|----------|------|
| UP 篇 | give up, wake up, clean up, pick up, show up | 5 |
| DOWN 篇 | sit down, calm down, break down, turn down, write down | 5 |
| OUT 篇 | find out, work out, run out of, figure out, check out | 5 |
| OFF 篇 | take off, turn off, put off, show off, pay off | 5 |
| ON 篇 | put on, turn on, get on, carry on, go on | 5 |
| IN/OVER/BACK 篇 | give in, come over, call back, get back, think over | 5 |
| **总计** | | **30** |
