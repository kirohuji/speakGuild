# 并列与转折 · 因果与让步 · 从句连接

## 学习包概览

本学习包是 **语法专项系列** 的组成部分，聚焦英语连接词（connectors）的系统学习。

### 适用对象

- **难度范围**：L2（中级基础）~ L3（中级提高）
- **前置要求**：掌握基础语法，能理解简单句
- **目标学习者**：希望提升口语/写作连贯性的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 并列与转折 | and/but/or/so + however/whereas/yet | L2 |
| 因果与让步 | because/since/therefore + although/even though/despite | L2 |
| 从句连接 | when/if/unless + who/which/that 定语从句 | L3 |

### 学习目标

完成本包学习后，学习者能够：

1. 使用 and/but/or/so 进行基本的并列和转折表达
2. 运用 however/whereas/yet 进行更丰富的对比
3. 用 because/since/therefore 表达因果关系
4. 用 although/even though/despite 表达让步关系
5. 用 when/if/unless 引导时间和条件从句
6. 用 who/which/that 引导定语从句修饰名词
