# 不规则动词大全 · 120 个核心动词精练（L2 核心）

## 学习包概览

本学习包是 **语法专项系列** 的组成部分，聚焦英语 **120 个核心不规则动词**。**纯 L2（中级基础）** 定位，精选 **120+ 个高频不规则动词句块（chunks）**—— 每个句块在一个完整句子中呈现动词的过去时和完成时用法，帮助学习者在语境中掌握三形变化。

> 英语不规则动词的三种形式无法通过规则推导记忆，是中国学习者的核心痛点。本包按 **变化模式分组**（AAA / ABB / ABA / ABC / 特殊），每组集中练习，熟能生巧。

### 按变化模式分组

| 模式 | 规律 | 动词数 | 例词 |
|:----:|------|:------:|------|
| **AAA** | 三形相同 | 14 | put-put-put, cut-cut-cut |
| **ABB** | 过去式 = 过去分词 | 56 | bring-brought-brought, find-found-found |
| **ABA** | 过去分词 = 原形 | 4 | come-came-come, run-ran-run |
| **特殊** | 混合变化 | 6 | be-was/were-been, go-went-gone |
| **ABC** | 三形都不同 | 40 | write-wrote-written, begin-began-begun |
| **总计** | | **120** | |

### 句块（chunk）vs 句型（pattern）

本包包含 **两类数据**，分工不同：

| | 句块（chunk） | 句型（pattern） |
|--|-------------|----------------|
| **是什么** | 可直接背诵的 **完整句子**（含时态对比） | 带替换槽位的 **句型框架** |
| **作用** | 在实际语境中记动词三形 | 替换动词尝试不同时态 |
| **例子** | `He broke the window. It has been broken before.` | `He ___ the window. It has been ___ before.`（slot: broke / broken） |
| **对应文件** | `chunks.csv` | `sentence_patterns.csv` |

> **学习路径**：先背句块（语境中记三形）→ 再练句型（替换不同动词）→ 形成条件反射。

### 设计理念

每组语法模式一个场景，每个动词至少出现一次，用过去时 + 完成时对比句呈现。120 个动词 × 1 句块 = **120+ 句块**。

### 适用对象

- **难度**：L2（中级基础）
- **前置要求**：掌握一般现在时、一般过去时和现在完成时的基本概念
- **目标学习者**：希望彻底掌握不规则动词变化、不再混淆三形的学习者

### 学习目标

完成本包学习后，学习者能够：

1. **AAA 组**：掌握 14 个三形相同的不规则动词，不再在完成时中误加-ed
2. **ABB 组**：掌握 56 个过去式=过去分词的动词，区分不同元音变化子模式
3. **ABA 组**：掌握过去分词回归原形的动词
4. **特殊组**：掌握 be/do/go 等最常用动词的复杂变化
5. **ABC 组**：掌握 40 个三形都不同的核心动词，重点攻克 i-a-u 和 o-en 两大元音链
6. **时态切换**：在一般过去时和现在完成时之间自如切换，用对动词形式
