# 方位介词 · 指路与位置描述

## 学习包概览

本学习包是 **语法专项系列** 的组成部分，系统学习英语中表达方位和指路的词汇与句型。

### 适用对象

- **难度范围**：L2（中级基础）
- **前置要求**：掌握基本动词和名词
- **目标学习者**：需要描述位置或问路指路的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 方位介词 | in/on/at/under/behind + next to/between/opposite/across from | L2 |
| 指路与位置描述 | go straight/turn left/turn right + past/on your left/right | L2 |

### 学习目标

完成本包学习后，学习者能够：

1. 使用 in/on/at 区分内部、表面和位置点
2. 使用 under/behind 表达下方和后方
3. 使用 next to/between/opposite 表达相对位置关系
4. 用 go straight/turn left/turn right 给出方向指令
5. 用 past/on your left/right 描述完整路线
