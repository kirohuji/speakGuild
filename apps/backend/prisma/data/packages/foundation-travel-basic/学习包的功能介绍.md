# 基础⑦ · 将来时与情态动词 · 计划预测 · 能力请求 · 建议义务（L1~L2）

## 学习包概览

本学习包是 **Foundation 基础系列** 的 **第七包**。你将学习 **将来时（will / be going to）** 和 **情态动词（can / must / should / may）**——表达计划、预测、能力、请求、建议和义务。

> 💡 **将来时 + 情态动词** 在一起是因为它们都涉及"非现实"——还没发生的事（将来）和可能/应该/必须做的事（情态）。这些表达让你的英语从"描述现实"升级到"谈论可能性和计划"。

### 本包核心语法

| 语法点 | 说明 | 示例 |
|:------|:-----|:-----|
| **will + 动词** | 即时决定/预测 | I'll call you. It will rain. |
| **be going to + 动词** | 计划/迹象 | I'm going to travel. It's going to rain. |
| **can** | 能力/许可 | I can swim. Can I open the window? |
| **must** | 必须(主观) | I must finish this today. |
| **should** | 建议 | You should see a doctor. |
| **may / might** | 可能/正式许可 | It may rain. May I come in? |
| **have to** | 必须(客观) | I have to work tomorrow. |
| **mustn't / don't have to** | 禁止/不必 | You mustn't park here. / You don't have to come. |

### 适用对象

- **难度**：L1~L2
- **前置要求**：一般现在时 + 一般过去时
- **目标**：能用将来时谈论计划和预测，能用情态动词表达各种态度

## 场景与话题

| 场景 | 话题 | 核心句型 |
|:----|:-----|:---------|
| 🅰️ **计划与预测** | ① 我打算... | I'm going to... / I will... / I'll... |
| | ② 将会怎样 | It will be... / They're going to... / 预测 |
| 🅱️ **能力与请求** | ③ 我能/会... | I can... / Can I...? / I can't... |
| | ④ 你可以/必须... | You can... / You must... / You should... |
| 🅲 **建议与义务** | ⑤ 你应该/必须 | You should... / I have to... / I need to... |
| | ⑥ 可能/不必/禁止 | It may... / You don't have to... / You mustn't... |
