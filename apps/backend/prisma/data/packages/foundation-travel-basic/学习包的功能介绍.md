# 基础·旅行基础100句

## 学习包概览

> 本学习包是 **基础口语系列** 的核心组成部分，专为英语输出训练设计。

### 适用对象

- **难度等级**：L1（零基础~初级）
- **目标学习者**：英语初学者，需要大量口语输出练习的学习者
- **学习方式**：跟读模仿 → 替换练习 → 自主输出

### 包含场景

- **基础·问路与交通** — 学习在外国问路和乘坐交通工具的基础表达。
- **基础·餐厅与购物** — 学习在餐厅点餐和商店购物的基础表达。

### 训练主题

- 问路指路（学习问路和指路的基本表达。）
- 乘坐交通（学习乘坐公交地铁出租车的表达。）
- 餐厅点餐（学习在餐厅点餐的常用表达。）
- 购物逛街（学习购物时的常用表达。）
