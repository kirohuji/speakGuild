# 《漫语町》日常社交篇：场景与英语学习内容设计文档

> 版本：v0.1  
> 用途：用于英语学习 App / 视觉小说式口语练习内容规划  
> 核心方向：用户扮演在海外生活的年轻人，完成咖啡店点餐、超市购物、朋友聚会等日常社交场景，训练**日常生活英语表达能力**。
