# 日常社交全场景 · 咖啡店 · 餐厅 · 超市 · 服装购物 · 朋友邀约 · 休闲娱乐 · 拜访做客（L1~L3 递进）

## 学习包概览

本学习包是 **场景日常系列** 的核心组成部分，围绕 **7 大日常社交功能场景** 展开——从咖啡店点单、餐厅用餐、超市购物，到服装百货、朋友邀约、休闲娱乐、拜访做客，覆盖一个人在海外生活**与社交相关的所有日常沟通需求**。

> ❌ 传统做法：social = 固定几个场景，内容浅、数量少（仅 22 个句块）。
> ✅ 本包做法：social = 日常人际交往全场景。**喝咖啡时说**、**吃饭时说**、**逛街时说**、**约朋友时说**、**做客时说**——每种社交场合都能自然开口。

在每个场景内，按 **L1→L2→L3** 的语言难度递进排列句块，让学习者清晰看到：**同一个社交需求，如何从最简单的表达一步步升级为更地道、更完整的说法。**

### 为什么按功能场景递进？

| 传统做法 | ✅ 本包按功能递进 |
|---------|----------------|
| 只覆盖咖啡店/超市/聚会 3 个场景 | 覆盖 7 大场景，完整社交生活 |
| 句子随机排列，看不出难度 | 同一功能下 L1→L2→L3 清晰标识 |
| 找不到自己水平对应的内容 | 从 L1 试起，能掌握就挑战下一级 |
| 只会一种说法，不会升级 | 同个意思学会多种说法，灵活切换 |

### 句块（chunk）vs 句型（pattern）

本包包含 **两类数据**，分工不同：

| | 句块（chunk） | 句型（pattern） |
|--|-------------|----------------|
| **是什么** | 可直接背诵的 **完整句子** | 带替换槽位的 **句型框架** |
| **作用** | 背下来直接用，培养语感 | 举一反三，替换关键词生成新句子 |
| **例子** | `I'd like a latte, please.` | `I'd like a ___, please.`（slot: latte / cappuccino / tea） |
| **对应文件** | `chunks.csv` | `sentence_patterns.csv` |

> **学习路径**：先按功能场景浏览句块→找到适合自己当前难度的句块背熟→用句型框架替换关键词举一反三→向上挑战更高难度。

### 句块设计原则

| 原则 | 说明 | ✅ 好句块 | ❌ 差句块 |
|------|------|----------|----------|
| **场景真实** | 海外生活每天会遇到的社交场景 | `Can I get separate checks?` | 生造不自然的句子 |
| **功能驱动** | 每句解决一个实际社交需求 | `Do you have this in a smaller size?` | 功能模糊的句子 |
| **递进清晰** | 同一功能下 L1→L2→L3 难度阶梯明显 | `Where is the milk?`→`Which aisle is the bread in?`→`Could you check the back for me?` | 全是同一难度 |
| **文化恰当** | 符合英语国家的社交习惯和礼貌 | `Thanks for having me! Should I take off my shoes?` | 不符合当地习惯 |

### 适用对象

- **难度**：L1（零基础）→ L3（中级提高），同一场景内渐进
- **前置要求**：无（L1 起点）
- **目标学习者**：任何需要在英语国家进行日常社交的学习者——从简单点餐购物，到深度社交往来
