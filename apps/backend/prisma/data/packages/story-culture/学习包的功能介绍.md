# 英美文化 · 全球文化 · 现代文化

## 学习包概览

本学习包是 **故事系列** 的组成部分，通过文化探索学习英语表达。

### 适用对象

- **难度范围**：L2-L3（中级）
- **前置要求**：掌握基础日常对话
- **目标学习者**：对跨文化交流感兴趣、希望了解世界多元文化的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 英国文化·传统与现代 | 王室、下午茶、体育娱乐 | L2 |
| 美国文化·节日与生活方式 | 节日、流行文化 | L2 |
| 全球文化·节日与习俗 | 亚洲传统、世界各地文化 | L2 |
| 现代文化·数字时代 | 社交媒体、当代社会议题 | L3 |

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 4 个文化场景 |
| `training_topics.csv` | 9 个训练主题 |
| `scene_vocabulary.csv` | 基础词汇表 |
| `chunks.csv` | 核心表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
