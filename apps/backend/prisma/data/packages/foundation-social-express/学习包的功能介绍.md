# 基础⑥ · 一般过去时 · 回忆经历 · 讲述故事 · 过去的状态和动作（L1~L2）

## 学习包概览

本学习包是 **Foundation 基础系列** 的 **第六包**。你将学习 **一般过去时（Past Simple）**——描述已经发生的事、过去的习惯、过去的经历。

> 💡 **为什么这是第六包？** 学会现在时（习惯/正在）之后，你需要能力说"昨天我做了……""上周我去了……"。一般过去时是叙述和讲故事的时态，是日常交流中不可或缺的。

### 本包核心语法

| 语法点 | 说明 | 示例 |
|:------|:-----|:-----|
| **be 动词过去式** | was / were | I was happy. They were at home. |
| **规则动词过去式** | 动词 + ed | I worked. She studied. They played. |
| **不规则动词过去式** | 30个核心不规则动词 | went / ate / had / did / saw / said / got / made |
| **否定句** | didn't + 动词原形 | I didn't go. She didn't eat. |
| **疑问句** | Did + 主语 + 动词原形 | Did you go? Did she eat? |
| **时间表达** | yesterday / last week / in 2020 / ago | I saw him yesterday. |

### 适用对象

- **难度**：L1~L2
- **前置要求**：一般现在时 + 现在进行时
- **目标**：能用过去时描述过去的经历和事件

## 场景与话题

| 场景 | 话题 | 核心句型 |
|:----|:-----|:---------|
| 🅰️ **过去的状态** | ① 我以前是/有 | I was... / I had... / He was... |
| | ② 昨天在哪里/怎么样 | I was at... / Were you...? / It was... |
| 🅱️ **过去的动作** | ③ 昨天做了什么 | I went / I ate / I watched / I played |
| | ④ 不规则动词 | went / ate / had / did / saw / said / got |
| 🅲 **讲述经历** | ⑤ 上个周末/去年 | Last weekend I... / Last year I... |
| | ⑥ 否定和疑问 | I didn't... / Did you...? / When did you...? |
