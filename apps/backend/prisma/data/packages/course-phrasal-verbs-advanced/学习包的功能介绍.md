# 进阶动词短语（L3~L4）

## 学习包概览

本学习包是 **语法专项系列** 的进阶组成部分，精选 **400 个中高级动词短语句块**。**L3~L4（中级提高~高级）** 定位，覆盖学术、商务、新闻、正式场合中的动词短语，与 `course-phrasal-verbs-daily`（L2）相衔接、不重复。

> 素材来源于邵艾伦原创整理的 600 常见词组，从中筛选 L3~L4 级别的条目。掌握这些短语后，学习者在阅读、听力、写作中都能显著提升理解与表达能力。

### 句块（chunk）vs 句型（pattern）

| | 句块（chunk） | 句型（pattern） |
|--|-------------|----------------|
| **是什么** | 可直接背诵的 **完整句子** | 带替换槽位的 **句型框架** |
| **作用** | 在语境中掌握进阶短语 | 替换宾语/人称，举一反三 |
| **例子** | `The issue boils down to one thing: money.` | `The issue boils down to ___.` |
| **对应文件** | `chunks.csv` | `sentence_patterns.csv` |

### 适用对象

- **难度**：L3~L4（中级提高~高级）
- **前置要求**：掌握 L2 日常动词短语
- **目标学习者**：希望在学术、商务、新闻场景中运用动词短语的学习者

### 包含场景

| 场景 | 短语数量 | 示例 |
|------|:--------:|------|
| 商务与职场 | ~50 | back up, break down, bring up, carry out, draw up |
| 新闻与媒体 | ~50 | blow up, break out, come about, crack down, die down |
| 学术与研究 | ~50 | boil down to, branch out, delve into, follow up, map out |
| 社会与法律 | ~50 | clamp down, close down, phase out, root out, stamp out |
| 情感与关系 | ~50 | fall apart, drift apart, grow apart, open up, bond with |
| 复杂动作/事件 | ~50 | pass away, break through, come across, end up, turn out |
| 正式表达① | ~50 | abide by, account for, adhere to, prevail over, refrain from |
| 正式表达② | ~50 | contend with, embark on, hinge on, partake in, vouch for |
| **总计** | **~400** | |

### 学习目标

完成本包学习后，学习者能够：

1. 在学术/商务/新闻场景中使用 400+ 个进阶动词短语
2. 理解英文新闻、播客、 TED 演讲中的高频短语动词
3. 在雅思/托福写作中使用恰当的短语动词提升分数
4. 区分同一短语在不同语境下的多种含义
