# 基础② · 一般动词与日常动作 · 饮食习惯 · 喜好需求 · 日程安排（L1）

## 学习包概览

本学习包是 **Foundation 基础系列** 的 **第二包**。你将在学完 be 动词的基础上，学习英语的 **一般动词（action verbs）**——也就是"做动作"的词：吃、喝、去、读、看、玩、喜欢、想要……

> 💡 **为什么这是第二包？** be 动词让你能说"我是谁""这是什么"，但生活的大部分交流是 about actions——"我吃早餐""我喜欢看书""我需要帮助"。一般动词 + 一般现在时，就是日常交流的引擎。

### 本包核心语法

| 语法点 | 说明 |
|:------|:-----|
| **一般动词现在时** | I eat / I go / I like（第一/二人称） |
| **否定句** | I don't...（don't + 动词原形） |
| **一般疑问句** | Do you...? + 简短回答 |
| **宾语代词** | me / you / him / her / it / us / them |
| **时间表达** | every day / in the morning / at night |
| **have/has 区别** | I have... / He/She has...（基础区分） |

### 和第一包的关系

| 第一包：be 动词 | 第二包：一般动词 |
|:---------------|:---------------|
| I am a student. | **I study** English. |
| It is big. | **It looks** big. |
| You are kind. | **You help** me. |
| 描述状态（是什么/怎么样） | 描述动作（做什么） |

### 适用对象

- **难度**：L1（零基础~初级）
- **前置要求**：掌握 be 动词（I am / You are / It is）
- **目标**：能用一般动词描述日常习惯、喜好和需求
