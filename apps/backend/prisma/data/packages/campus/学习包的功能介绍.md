# 《漫语町》校园生活篇：场景与英语学习内容设计文档

> 版本：v0.1  
> 用途：用于英语学习 App / 视觉小说式口语练习内容规划  
> 核心方向：用户扮演在校大学生，在校园内完成图书馆、社团、食堂、选课等日常事务，训练**校园场景的英语表达能力**。
