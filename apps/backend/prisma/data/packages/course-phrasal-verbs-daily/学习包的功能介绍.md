# 日常动词短语（L2 核心）

## 学习包概览

本学习包是 **语法专项系列** 中最实用的组成部分之一，精选 **200 个日常高频动词短语句块**。**纯 L2（中级基础）** 定位，覆盖日常交流中最常用的动词短语，每个句块一个完整句子，口语拿来就用。

> 素材来源于邵艾伦原创整理的 600 常见词组，从中筛选 L2 级别的日常高频短语。动词短语是英语母语者日常交流的核心——想听懂老外说话、说得像老外，必须掌握。

### 句块（chunk）vs 句型（pattern）

| | 句块（chunk） | 句型（pattern） |
|--|-------------|----------------|
| **是什么** | 可直接背诵的 **完整句子** | 带替换槽位的 **句型框架** |
| **作用** | 背下整句，脱口而出 | 替换宾语/人称，举一反三 |
| **例子** | `Calm down! Everything will be fine.` | `Calm down! ___` |
| **对应文件** | `chunks.csv` | `sentence_patterns.csv` |

### 适用对象

- **难度**：L2（中级基础）
- **前置要求**：掌握基本动词
- **目标学习者**：希望口语更地道、听懂日常对话的学习者

### 包含场景

| 场景 | 短语数量 | 示例 |
|------|:--------:|------|
| 日常生活（起床/出门/回家） | ~25 | wake up, get up, go out, come back, get home |
| 工作学习 | ~25 | fill in, find out, figure out, hand in, turn in |
| 人际关系 | ~25 | get along, hang out, cheer up, calm down, pick up |
| 交通出行 | ~25 | take off, get on, get off, pick up, drop off |
| 购物餐饮 | ~25 | try on, pay for, eat out, pick out, warm up |
| 电话/社交 | ~25 | call back, hang on, hold on, speak up, show up |
| 居家生活 | ~25 | clean up, tidy up, put away, turn off, throw away |
| 情绪/状态 | ~25 | give up, cheer up, slow down, settle down, burn out |
| **总计** | **~200** | |

### 学习目标

完成本包学习后，学习者能够：

1. 在日常会话中自然使用 200+ 个 L2 动词短语
2. 理解英语母语者日常对话中的动词短语
3. 将课本英语（discover → find out / tolerate → put up with）转换为地道表达
4. 避免动词短语使用时常见的语法错误（宾语位置）
