# 频率与程度副词 · 方式与评论副词

## 学习包概览

本学习包是 **语法专项系列** 的组成部分，系统学习英语副词的分类与用法。

### 适用对象

- **难度范围**：L2（中级基础）
- **前置要求**：掌握基本形容词和动词用法
- **目标学习者**：希望表达更精确、丰富的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 频率与程度副词 | always/usually/often/sometimes/never + very/quite/rather/extremely/too | L2 |
| 方式与评论副词 | carefully/quickly/happily/easily/well + luckily/frankly/honestly/surprisingly/obviously | L2 |

### 学习目标

完成本包学习后，学习者能够：

1. 使用频率副词 (always→never) 描述事件频率，掌握词序规则
2. 使用程度副词 (very→too) 精确表达程度强弱
3. 使用方式副词描述动作进行的方���
4. 使用评论副词对整句发表态度或评价
5. 区分不同类别副词在句中的位置规则
