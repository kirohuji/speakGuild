# 雅思·IELTS Band 8 优秀

## 学习包概览

本学习包是 **雅思口语专项系列** 的重要组成部分。

### 适用对象

- **难度等级**：L4
- **目标分数**：Band 8 优秀
- **前置要求**：能讨论抽象话题，词汇量较丰富
- **目标学习者**：备战雅思口语考试，目标Band 8 优秀分的考生

### 包含主题

| 主题 | 类型 | 时长 |
|------|:----:|:----:|
| 难忘的经历 | P1/P2/P3 口语题 | 90秒 |
| 教育与社会 | P1/P2/P3 口语题 | 75秒 |

### 核心词汇

- **memorable** (难忘的) — 值得记忆的、难忘的。
- **perspective** (视角) — 观点、视角。
- **profound** (深刻的) — 深刻的、意义深远的。
- **critical** (批判性的) — 批判的、关键的。

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 场景定义 |
| `training_topics.csv` | 2 个训练主题 |
| `scene_vocabulary.csv` | 4 个核心词汇 |
| `chunks.csv` | 核心表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 2 个学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
