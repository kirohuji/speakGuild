# 托福 · 学术讨论 / 综合口语

## 学习包概览

本学习包是 **托福专项** 学习材料。

### 适用对象

- **难度等级**：L3
- **目标学习者**：准备托福考试的学习者

### 包含场景

- **托福·学术讨论** — TOEFL Speaking Task 1-2：校园场景、学术讨论。掌握托福口语评分标准和答题框架。
- **托福·综合口语** — TOEFL Speaking Task 3-4：综合阅读、听力与口语。掌握笔记技巧和信息整合表达。

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 2 个场景 |
| `training_topics.csv` | 4 个训练主题 |
| `scene_vocabulary.csv` | 核心词汇 |
| `chunks.csv` | 表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
