# 《漫语町》健康医疗篇：角色、场景、故事线与英语学习内容设计文档

> 版本：v0.1  
> 用途：用于英语学习 App / 视觉小说式口语练习内容规划  
> 核心方向：用户扮演在海外生活的年轻人，从预约看病到药房取药，覆盖**海外就医的完整英语表达场景**。  
> 视觉基调：干净、柔和、浅蓝/白色调、医疗场景感、线条简洁、角色 7 分立绘 + 1:1 头像 + 9:16 场景图。
>
> ⚠️ **范围说明**：本包专注**日常就医**场景——预约、看诊、买药、急诊。不包含住院手术、医疗保险条款等复杂场景（可后续扩展）。

---

## 0. 总体定位

《漫语町》这一章节可以叫作：

**Healthcare English / 健康医疗篇**

核心不是"背医学词汇"，而是让用户在真实的海外就医场景中，学会如何表达自己的病情、理解医生的诊断、在药房正确买药。

用户会从"不舒服但不知道怎么开口"开始，逐步学会：

- 打电话预约诊所
- 描述症状（头痛、发烧、过敏、拉肚子...）
- 回答医生问诊（何时开始、吃过什么药、有没有过敏史）
- 理解诊断和治疗方案
- 在药房买药、理解用法用量
- 紧急情况下去急诊求助

这一篇的主线是：

> **在国外生病了，我知道怎么去看医生、怎么说清楚。**

# 1. 角色设计

## 1.1 主人公：Liam

| 项目 | 设定 |
|---|---|
| 身份 | 在海外生活的年轻人 / 留学生 / 工作人士 |
| 性格 | 独立、但有点害怕看医生（语言障碍） |
| 视觉特征 | 简洁休闲装、略显紧张的表情、有时裹着围巾（生病） |
| 主要功能 | 承接用户视角，让用户代入他进行就医英语输出 |
| 对话服务 | 预约、看诊、买药、急诊 |

### Liam 服务的语言目标

常见句型：

```text
I'd like to make an appointment.
I've been having a headache for a few days.
I have a fever and a sore throat.
How many times a day should I take this?
```

常见词汇：

```text
appointment, symptom, prescription, pharmacy, fever, cough, headache, allergy, dosage, side effect
```

---

## 1.2 Dr. Williams：家庭医生

| 项目 | 设定 |
|---|---|
| 身份 | General Practitioner (GP) / 全科医生 |
| 性格 | 耐心、温和、解释清楚、不赶时间 |
| 视觉特征 | 白大褂、听诊器、温和微笑 |
| 主要功能 | 引导用户完成看诊对话 |

### Dr. Williams 服务的语言目标

常见句型：

```text
What seems to be the problem?
How long have you had these symptoms?
Have you taken any medication?
I'll prescribe some antibiotics.
Come back if it doesn't get better in a week.
```

# 5. 总结

v0.1 健康医疗篇覆盖**预约→看诊→买药**三大海外就医核心阶段：

1. **实用导向**：每个 Topic 解决一个真实就医场景中的英语任务（40 Chunk × 44 Vocab × 27 Pattern）
2. **由浅入深**：从电话预约到理解诊断，难度逐步递进（L1→L3）
3. **可扩展**：后续可加急诊、牙科、专科、住院等章节

> **在国外生病了，我知道怎么去看医生、怎么说清楚。**
