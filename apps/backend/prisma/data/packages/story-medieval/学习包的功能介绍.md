# 中世纪·骑士与城堡 · 探险与传奇

## 学习包概览

本学习包是 **故事系列** 的组成部分，通过中世纪冒险故事学习英语。

### 适用对象

- **难度范围**：L2（中级基础）
- **前置要求**：掌握基本过去时
- **目标学习者**：对中世纪历史、骑士传说和冒险故事感兴趣的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 中世纪·骑士与城堡 | 骑士精神、城堡攻防、亚瑟王传说 | L2 |
| 中世纪·探险与传奇 | 马可·波罗、维京航海家、中世纪世界 | L2 |

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 2 个中世纪场景 |
| `training_topics.csv` | 6 个训练主题 |
| `scene_vocabulary.csv` | 基础词汇表 |
| `chunks.csv` | 核心叙事例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
