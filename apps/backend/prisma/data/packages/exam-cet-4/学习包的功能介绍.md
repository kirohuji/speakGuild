# 大学英语 · 校园生活 / 社会热点

## 学习包概览

本学习包是 **大学英语专项** 学习材料。

### 适用对象

- **难度等级**：L2
- **目标学习者**：准备大学英语考试的学习者

### 包含场景

- **四级·校园生活** — CET-4常考话题：校园生活、选课、社团活动。掌握四级核心词汇和基本议论文表达。
- **四级·社会热点** — CET-4写作和翻译常考话题：环保、科技、文化。掌握四级高频词汇和议论文结构。

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 2 个场景 |
| `training_topics.csv` | 4 个训练主题 |
| `scene_vocabulary.csv` | 核心词汇 |
| `chunks.csv` | 表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
