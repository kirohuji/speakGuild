# 雅思·IELTS Band 7 良好

## 学习包概览

本学习包是 **雅思口语专项系列** 的重要组成部分。

### 适用对象

- **难度等级**：L3
- **目标分数**：Band 7 良好
- **前置要求**：能进行较流利的日常对话
- **目标学习者**：备战雅思口语考试，目标Band 7 良好分的考生

### 包含主题

| 主题 | 类型 | 时长 |
|------|:----:|:----:|
| 推荐旅游地 | P1/P2/P3 口语题 | 90秒 |
| 城市发展 | P1/P2/P3 口语题 | 60秒 |

### 核心词汇

- **scenic** (风景优美的) — 景色优美的。
- **recommend** (推荐) — 推荐。
- **attract** (吸引) — 吸引。
- **diverse** (多样化的) — 多元的、多样的。

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 场景定义 |
| `training_topics.csv` | 2 个训练主题 |
| `scene_vocabulary.csv` | 4 个核心词汇 |
| `chunks.csv` | 核心表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 2 个学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
