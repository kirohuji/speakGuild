# 大学英语 · 社会发展 / 文化教育

## 学习包概览

本学习包是 **大学英语专项** 学习材料。

### 适用对象

- **难度等级**：L3
- **目标学习者**：准备大学英语考试的学习者

### 包含场景

- **六级·社会发展** — CET-6高频话题：城市化、人口、经济发展。掌握六级核心词汇和深度议论文表达。
- **六级·文化教育** — CET-6写作翻译常考：文化交流、教育改革。掌握六级高级词汇和论证技巧。

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 2 个场景 |
| `training_topics.csv` | 4 个训练主题 |
| `scene_vocabulary.csv` | 核心词汇 |
| `chunks.csv` | 表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
