# 《漫语町》旅行英语篇：角色、场景、故事线与英语学习内容设计文档

> 版本：v0.1  
> 用途：用于英语学习 App / 视觉小说式口语练习内容规划  
> 核心方向：用户扮演旅行者，从机场出发开始，完整经历从出发到回程的旅行全流程，聚焦**独立出行的英语表达能力**。  
> 视觉基调：清爽、明快、淡蓝天空色/暖米色、开阔场景感、线条轻盈、旅行向、角色 7 分立绘 + 1:1 头像 + 9:16 场景图。
>
> ⚠️ **范围说明**：留学生活、职场办公等长期生活类场景**不属于本学习包**。本包专注**短期旅行**场景：机场、酒店、交通、餐饮、购物、应急。

---

## 0. 总体定位

《漫语町》这一章节可以叫作：

**Travel English / 旅行英语篇**

核心不是"背旅行词汇"，而是让用户通过一个完整的旅行故事，反复练习旅行中最常遇到的英语输出场景。

用户会从"准备出发"开始，逐步学会：

- 机场值机、安检、登机、中转
- 入境回答海关问题
- 酒店入住、客房服务、退房
- 打车、问路、乘公交地铁
- 餐厅点餐、预约、结账
- 购物询价、退税、投诉
- 丢失物品、生病求助等应急情况

这一篇的主线是：

> **从出发到回程，我能独立完成整个旅行。**

# 1. 角色设计

## 1.1 主人公：Maya

| 项目 | 设定 |
|---|---|
| 身份 | 第一次出国的年轻旅行者 / 自由行游客 |
| 性格 | 勇敢、有点小紧张、热爱探索 |
| 视觉特征 | 双肩包、旅行箱、相机、舒适休闲装 |
| 主要功能 | 承接用户视角，让用户代入她进行旅行英语输出 |
| 对话服务 | 机场、酒店、餐厅、购物、问路、求助 |
| 为什么需要 | 用户需要一个"从紧张到自如"的旅行代入角色 |

### Maya 服务的语言目标

常见句型：

```text
I'd like to check in for my flight.
Could you help me find this place?
I have a reservation.
Can I have the menu, please?
How much does this cost?
I lost my bag.
```

常见词汇：

```text
flight, boarding pass, luggage, reservation, menu, receipt, exchange rate, direction, platform, emergency
```

# 3. 故事线设计

## 3.1 整体故事主题

主人公 Maya 第一次出国自由行。她从出发去机场就开始紧张——英语怎么说、值机怎么办理、入境怎么回答…… 随着她一步步走过：

1. 机场值机 → 2. 飞机体验 → 3. 入境通关 → 4. 打车去酒店 → 5. 酒店入住 → 6. 出门问路 → 7. 餐厅吃饭 → 8. 购物逛街 → 9. 应急处理 → 10. 退房回家

她发现——原来自己可以做到。

故事核心是：

> **把旅行全过程拆成一个个可练习的英语场景。**

# 6. 总结

v0.1 旅行英语篇覆盖**出发→抵达→出行→住宿→街头**五大旅行核心场景：

1. **全流程覆盖**：从机场值机到街头问路，12 个 Topic 覆盖旅行必经场景（5 Scene × 12 Topic × 41 Chunk × 43 Vocab × 24 Pattern）
2. **实用导向**：每个 Topic 解决一个真实旅行中的英语任务
3. **渐进难度**：Ch1/L1-L2（机场）→ Ch2/L1-L2（入境）→ Ch3/L1-L2（交通）→ Ch4/L1-L2（住宿）
4. **可扩展**：后续可加餐饮、购物、应急等章节

> **遇到一个真实旅行场景时，我知道怎么开口。**
