# 科技发展史 · 数字革命与未来

## 学习包概览

本学习包是 **故事系列** 的组成部分，通过科技发展故事学习英语。

### 适用对象

- **难度范围**：L2-L3（中级）
- **前置要求**：掌握基本科学词汇
- **目标学习者**：对科技感兴趣、希望用英语讨论科技话题的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 科技·改变世界的发明 | 印刷术、蒸汽机、电力与通信 | L2 |
| 科技·数字革命与未来 | 计算机互联网、人工智能与未来科技 | L3 |

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 2 个科技场景 |
| `training_topics.csv` | 5 个训练主题 |
| `scene_vocabulary.csv` | 基础词汇表 |
| `chunks.csv` | 核心表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |

### 关键词

`科技` `发明` `数字革命` `AI` `太空探索` `technology` `innovation`
