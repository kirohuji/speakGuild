# 街头巷尾 · 外卖 · 出行 · 邻里 · 小店（L1~L2）

## 学习包概览

本学习包是 **日常系列** 的街头生活场景包。**核心定位是贴近真实生活场景**——外卖取快递、扫码坐公交、便利店小摊、邻里寒暄——这些都是日常生活中真正会用到的场景。不假设"人在国外"，只假设"用英语处理身边的事"。

### 包含场景（6 大场景 × 18 话题）

| 场景名称 | 话题 | 难度 | 句块数 |
|---------|------|:----:|:------:|
| 📱 外卖与网购 | 点外卖、退换货、客服沟通 | L1 | ~8 |
| 🚄 出行路上 | 网约车、高铁、扫码支付 | L1~L2 | ~8 |
| 🏘️ 社区邻里 | 取快递、物业报修、邻里交流 | L2 | ~8 |
| 🏪 街边小店 | 便利店、小吃摊、菜市场 | L1 | ~8 |
| 🤝 街头社交 | 搭讪问路、朋友约饭、团建聚会 | L2 | ~8 |
| 🛵 生活服务 | 跑腿代办、维修报修、社区通知 | L2 | ~6 |
