# 基础⑧ · 连接与表达 · 并列转折 · 因果条件 · 时间顺序 · 举例补充 · 综合叙事（L1）

## 学习包概览

本学习包是 **基础系列（Foundation）· 第 8 包**，也是从"说短句"到"说段落"的 **表达升级包**。前七个包你学会了各种时态和情态动词——现在你要学会把这些句子**连接起来**，表达因果、转折、时间、条件等逻辑关系。

> 🎯 **核心目标**：学完本包，你能用连词把简单句连接成复杂句，按时间顺序叙事、解释原因和结果、表达条件和假设——**从"碎片短句"到"流畅段落"。**

### 为什么连接词需要专门一个包？

有了前面的基础，你已经能说"I went to the store"、"I bought some milk"、"I came home"。但真正的交流需要连接这些想法：

- ❌ **碎片式**：I went to the store. I bought some milk. I came home.
- ✅ **流畅式**：I **went** to the store, **bought** some milk, **and** **then** I came home.

中文靠"意合"——把短句放一起就行。英语靠"形合"——**必须用连词显式表达逻辑关系**。

> ❌ 中国人常犯错误：*I like coffee, I like tea too. / *I went home I was tired. / *If it will rain, I'll stay home.
> ✅ 正确：**I like coffee and tea. / I went home because I was tired. / If it rains, I'll stay home.**

## 设计哲学

### 两层词汇分配策略

本包是**连词/连接词驱动型**学习包，核心词汇分为两类：

| 层级 | 来源 | 作用 | 本包占比 |
|------|------|------|:--------:|
| **核心词汇** | 包⑧新引入（包①-⑦未覆盖） | 连词本身 + 多词连接短语 | 112 词/短语 |
| **扩展词汇** | 复用包①-⑦ | 被连接的内容实义词 | ~140 词 |

> 🧠 **核心洞察**：连词是英语的"逻辑胶水"。中文靠意合（短句堆砌），英语靠形合（必须显式标注逻辑关系）。本包的 112 个核心词/短语中，约 70% 是**多词连接短语**（如 not only...but also、as long as、by the way），约 30% 是**单个连词或逻辑副词**（如 and/but/yet/thus/hence/furthermore）。这些连接词本身数量不多，但它们是英语从"碎片短句"升级到"流畅段落"的关键。

### 累加式设计

包①-⑦覆盖了时态、情态、疑问等句子内部结构，本包引入句子之间的逻辑连接：

```
包①-③ 基础时态 → 包④ 短语动词 → 包⑤ 观点表达 → 包⑥ 社交表达 → 包⑦ 将来时+情态
                                    ↓
                              包⑧ 连词与逻辑连接（从"说短句"到"说段落"）
```

### 连词驱动词汇分配

每个 Topic 的核心词汇围绕一个**逻辑连接功能**组织：

| 逻辑功能 | 核心连词类型 | 示例 |
|----------|------------|------|
| 并列 | and / both...and / not only...but also | I like coffee and tea. |
| 转折 | but / yet / nevertheless / whereas | I like him, but he doesn't like me. |
| 因果 | because / since / thus / hence / therefore | I stayed home because I was sick. |
| 条件 | if / unless / as long as / provided that | If it rains, I'll stay home. |
| 时间顺序 | when / while / before / after / until / first...then...finally | When I arrived, she was cooking. |
| 对话衔接 | well / actually / by the way / anyway / I mean | Well, I think it's a good idea. |

### 多词短语优先

英语的连接逻辑大量依赖**多词固定短语**（如 as long as、in addition to、by the way、on the other hand），这些短语必须作为整体记忆，不宜拆词教学。因此本包核心行包含较多多词条目。

## 设计清单

### 常见分配错误

| 错误类型 | 错误示例 | 正确做法 |
|----------|---------|---------|
| because 和 so 叠用 | ❌ Because I was tired, so I went home. | ✅ Because I was tired, I went home. / I was tired, so I went home. |
| if 从句用 will | ❌ If it will rain, I'll stay home. | ✅ If it rains, I'll stay home. |
| 把基础连词当核心词 | ❌ and/or/but/so/if/when 标为核心 | ✅ 这些在包①已出现，应标为扩展 |
| 拆散固定连接短语 | ❌ 把 "as long as" 拆成 as + long + as | ✅ 作为整体多词短语教学 |
| 忽略逻辑副词 | ❌ 只教 and/but/because，不教 thus/hence/moreover | ✅ 逻辑副词是正式表达的进阶组件 |
| 连词前后结构不一致 | ❌ I like swimming and to run. | ✅ I like swimming and running. / I like to swim and run. |

### 🅰️ 并列与转折（5 主题，14 句块）

#### 第 1 课：和/或者/但是

| 主题 | 句块 | 释义 | 替换练习 |
|:----|:-----|:-----|:---------|
| and 并列 | **I like coffee and tea.** | 我喜欢咖啡和茶。 | I like ___ and ___.（cats and dogs / reading and writing / music and art） |
| and 并列 | **She is smart and kind.** | 她聪明又善良。 | He is ___ and ___.（tall and handsome / young and energetic / funny and friendly） |
| and 连接动作 | **I went to the store and bought some milk.** | 我去了商店，买了些牛奶。 | I ___ and ___.（came home and cooked dinner / sat down and read a book） |
| or 选择 | **Do you want tea or coffee?** | 你想喝茶还是咖啡？ | Do you want ___ or ___?（this or that / to stay or to go / pizza or pasta） |
| or 选择问 | **Is it big or small?** | 它是大的还是小的？ | Is it ___ or ___?（hot or cold / new or old / cheap or expensive） |
| but 转折 | **I like him, but he doesn't like me.** | 我喜欢他，但他不喜欢我。 | I ___, but ___.（want to go, but I'm busy / tried my best, but I failed） |
| but 对比 | **The food is cheap, but it's delicious.** | 食物便宜，但是很好吃。 | ___ is ___, but ___.（the hotel is small, but it's clean / he's young, but very smart） |

#### 第 2 课：因为/所以

| 主题 | 句块 | 释义 | 替换练习 |
|:----|:-----|:-----|:---------|
| because 原因 | **I stayed home because I was sick.** | 我待在家，因为我病了。 | I ___ because ___.（was late because I missed the bus / ate a lot because I was hungry） |
| because 原因 | **She is happy because she passed the test.** | 她开心，因为她通过了考试。 | ___ because ___.（He's tired because he worked all night / We're excited because it's our first trip） |
| Because 前置 | **Because it rained, we stayed inside.** | 因为下雨了，我们待在了屋里。 | Because ___, ___.（it was cold, I wore a jacket / I was tired, I went to bed early） |
| so 结果 | **I was tired, so I went to bed early.** | 我累了，所以我早早上床了。 | I was ___, so I ___.（hungry, so I ate / busy, so I couldn't come / bored, so I watched TV） |
| so 结果 | **It started to rain, so we took a taxi.** | 开始下雨了，所以我们打了车。 | ___, so ___.（the store was closed, so I went home / I lost my phone, so I bought a new one） |
| however 对比 | **I wanted to go. However, I was too busy.** | 我想去。然而，我太忙了。 | I wanted to ___. However, ___.（come / I was sick / stay / I had to work） |
| not only...but also | **She not only speaks English, but also speaks Chinese.** | 她不仅说英语，还说中文。 | He not only ___ but also ___.（sings / dances / can cook / can bake / likes music / likes art） |

### 🅱️ 因果与条件（5 主题，14 句块）

#### 第 3 课：当…时/之前/之后/直到

| 主题 | 句块 | 释义 | 替换练习 |
|:----|:-----|:-----|:---------|
| when 当…时 | **When I arrived, she was cooking.** | 我到的时候，她正在做饭。 | When I ___, ___.（called, he was sleeping / got home, they were watching TV） |
| when 习惯 | **I always listen to music when I run.** | 我跑步的时候总是听音乐。 | I always ___ when I ___.（drink coffee when I wake up / read when I take the bus） |
| when 将来 | **Call me when you get home.** | 你到家的时候给我打电话。 | ___ when you ___.（text me when you arrive / tell her when you see her） |
| before 之前 | **I brush my teeth before I go to bed.** | 我睡觉前刷牙。 | I ___ before I ___.（eat breakfast before I leave / check my email before I start work） |
| after 之后 | **After I finished work, I went to the gym.** | 我下班后去了健身房。 | After I ___, I ___.（woke up, I took a shower / ate dinner, I watched a movie） |
| while 同时 | **We played games while we waited.** | 我们等的期间玩了游戏。 | I ___ while ___.（listened to music while I worked / talked on the phone while I walked） |
| until 直到 | **Please wait here until I come back.** | 请在这里等到我回来。 | ___ until ___.（stay here until it stops raining / keep trying until you succeed） |

#### 第 4 课：如果/除非/只要

| 主题 | 句块 | 释义 | 替换练习 |
|:----|:-----|:-----|:---------|
| if 条件 | **If it rains tomorrow, I'll stay home.** | 如果明天下雨，我就待在家。 | If ___, I'll ___.（I have time, I'll call you / you're free, we can meet / it's cheap, I'll buy it） |
| if 条件 | **If you study hard, you will pass the test.** | 如果你努力学习，你会通过考试的。 | If you ___, you will ___.（practice every day, get better / eat well, feel better / save money, be rich） |
| 祈使+if | **Let me know if you need help.** | 如果你需要帮助就告诉我。 | ___ if ___.（call me if you get lost / take an umbrella if it rains / tell her if you see her） |
| unless 除非 | **I won't go unless you come with me.** | 除非你跟我一起去，否则我不去。 | I won't ___ unless ___.（eat unless I'm hungry / leave unless you ask me to / buy it unless it's cheap） |
| unless 禁止 | **You can't enter unless you have a ticket.** | 除非你有票，否则你不能进。 | You can't ___ unless ___.（drive unless you have a license / pass unless you study / go out unless you finish） |
| 虚拟建议 | **If I were you, I would take the job.** | 如果我是你，我会接受那份工作。 | If I were you, I would ___.（say sorry / go to the doctor / study abroad / wait a bit longer） |
| as long as 只要 | **As long as you try your best, it's okay.** | 只要你尽力了，就没关系。 | As long as ___, ___.（you're happy, I'm happy / we're together, everything is fine / it doesn't rain, we'll go） |

### 🅲 时间与顺序（5 主题，13 句块）

#### 第 5 课：举例与补充

| 主题 | 句块 | 释义 | 替换练习 |
|:----|:-----|:-----|:---------|
| 举例 | **I like sports. For example, I play basketball.** | 我喜欢运动。比如，我打篮球。 | I like ___. For example, I ___.（music / listen to pop / movies / watch comedies / food / love hot pot） |
| 补充 | **She is kind. Also, she is very hardworking.** | 她很善良。而且，她非常勤奋。 | He is ___. Also, he is ___.（smart / funny / tall / strong / rich / generous / young / brave） |
| too | **I like coffee. He likes coffee, too.** | 我喜欢咖啡。他也喜欢咖啡。 | I ___. He ___, too.（like tea / likes tea / play guitar / plays guitar / am from Beijing / is from Beijing） |

| what's more | **It's a nice restaurant. What's more, the prices are good.** | 这家餐厅不错。而且，价格也实惠。 | It's a ___. What's more, ___.（good movie / it's free / nice hotel / it's near the beach / fun game / it's easy to learn） |
| besides | **I want to go, but I have no time. Besides, I don't have money.** | 我想去，但我没时间。再说，我也没钱。 | I want to ___, but ___. Besides, ___.（travel / I'm busy / tickets are expensive / eat out / I'm tired / I need to save money） |

| 主题 | 句块 | 释义 | 替换练习 |
|:----|:-----|:-----|:---------|

### 🅳 对话衔接（5 主题，13 句块）

#### 第 7 课：对话衔接词

> 🧠 **口语润滑剂**：well, actually, by the way, anyway — 这些词不改变句子意思，但让你的英语听起来**自然流畅**。

| 主题 | 句块 | 释义 | 使用场景 |
|:----|:-----|:-----|:---------|
| Well | **Well, I think it's a good idea.** | 嗯，我觉得这是个好主意。 | 回答前稍作停顿 |
| Actually | **Actually, I don't like coffee. I prefer tea.** | 其实，我不喜欢咖啡。我更喜欢茶。 | 纠正/补充之前的说法 |
| By the way | **By the way, did you hear about the party?** | 顺便问一下，你听说那个派对了吗？ | 切换话题或想起别的事 |
| Anyway | **Anyway, let's get back to the topic.** | 不管怎样，我们回到正题吧。 | 结束跑题/回到主题 |
| I mean | **I mean, it's not that I don't want to go. I'm just tired.** | 我是说，不是我不想去。我只是累了。 | 解释/澄清自己的意思 |
| Let me see | **Let me see... I think the answer is B.** | 让我想想……我觉得答案是 B。 | 需要时间思考 |
| You know | **You know, I've always wanted to learn guitar.** | 你知道的，我一直想学吉他。 | 引起共鸣或分享感受 |

## 附：连词与连接速查卡

```
📐 连词按功能分类

  并列连接   → and / or / but / however / not only...but also
  因果连接   → because / so / since / therefore / as a result
  时间连接   → when / while / before / after / until / as soon as
  条件连接   → if / unless / as long as / provided that
  举例补充   → for example / also / too / as well / besides
  顺序连接   → first / then / after that / next / finally

💡 连词位置速记：

  句首连词（逗号分隔两句话）：
    However, ... / Therefore, ... / As a result, ... / First, ...
    → However, I changed my mind. / First, I took a shower.

  句中连词（连接两个分句）：
    ... because ... / ... so ... / ... but ... / ... and ...
    → I stayed home because I was sick. / I was tired, so I went home.

  句末补充词：
    ... too / ... as well / ... either / ... though
    → I like it too. / I don't like it either.

⚠️ 最容易犯的错误：
  ❌ If it will rain, I'll stay home.   → if 从句不用 will
  ✅ If it rains, I'll stay home.
  
  ❌ I like drink coffee and tea.        → and 前后结构要一致
  ✅ I like coffee and tea. / I like to drink coffee and tea.
  
  ❌ Because I was tired, so I went home.  → because 和 so 只能用一个
  ✅ Because I was tired, I went home. / I was tired, so I went home.
  
  ❌ I not only speak English but also Chinese.  → not only...but also 后接平行结构
  ✅ I speak not only English but also Chinese.
```

## 输出训练流水线（Warmup Pipeline）

每个训练主题配有 `warmup_pipeline.json` 输出训练流水线，包含题型：

| 题型 | 说明 | 示例 |
|:----|:-----|:-----|
| **conjunction_combine** | 用连词合并两个短句 | I was tired. I went home. → I was tired, so I went home. |
| **conjunction_choice** | 根据逻辑选择正确连词 | I stayed home ___ I was sick. → (and/but/because/so) → because |
| **sentence_chaining** | 按顺序连接多个事件 | First...Then...After that...Finally... |
| **narrative_link** | 用因果/条件链叙述一件事 | Because...so... / If...then... |
| **discourse_marker** | 在对话中插入合适的衔接词 | ___, I think it's a good idea. → Well |

覆盖全部 25 个训练主题，L1 难度，重点是把简单句连接成复杂句和叙事链。
