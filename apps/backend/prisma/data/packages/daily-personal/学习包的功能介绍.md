# 个人内在生活 · 阅读 · 独处 · 反省 · 思考 · 放松 · 兴趣 · 成长（L1~L3 递进）

## 学习包概览

本学习包是 **场景日常系列** 的全新组成部分，聚焦一个人的 **内在世界与个人生活**——那些不需要别人参与、自己与自己相处的时刻。**阅读、独处、反省、思考、放松、兴趣、个人成长**，覆盖一个人在海外生活**与自己相处的所有日常沟通需求**。

> 其他 daily 包（campus / healthcare / social）解决的是"和外界打交道"的场景，而本包解决的是"**和自己相处**"的场景——怎么表达你的阅读习惯、怎么描述你的独处时光、怎么聊聊你的爱好和人生思考。

在每个场景内，按 **L1→L2→L3** 的语言难度递进排列句块，让学习者清晰看到：**同一个人话题，如何从最简单的表达一步步升级为更丰富、更有深度的说法。**

### 为什么需要"个人生活"英语？

| 场景 | 你可能想用英语说... |
|------|-------------------|
| 📚 阅读时 | `I'm reading a book about psychology.` / `I read before bed every night.` |
| 🧘 独处时 | `I need some me-time.` / `I enjoy my own company.` |
| 💭 反省时 | `I keep a journal to reflect on my day.` / `I've been thinking a lot about my future.` |
| 🎨 发展兴趣 | `I've started learning photography.` / `I made this myself!` |
| 🌱 个人成长 | `I set small goals each week.` / `I believe in continuous improvement.` |

### 句块（chunk）vs 句型（pattern）

本包包含 **两类数据**，分工不同：

| | 句块（chunk） | 句型（pattern） |
|--|-------------|----------------|
| **是什么** | 可直接背诵的 **完整句子** | 带替换槽位的 **句型框架** |
| **作用** | 背下来直接用，培养语感 | 举一反三，替换关键词生成新句子 |
| **例子** | `I'm currently reading a book about psychology.` | `I'm currently reading a book about ___.`（slot: psychology / history / fiction） |
| **对应文件** | `chunks.csv` | `sentence_patterns.csv` |

> **学习路径**：先按功能场景浏览句块→找到适合自己当前难度的句块背熟→用句型框架替换关键词举一反三→向上挑战更高难度。

### 句块设计原则

| 原则 | 说明 | ✅ 好句块 | ❌ 差句块 |
|------|------|----------|----------|
| **内在真实** | 真实反映一个人的内心世界和独处时光 | `I've been journaling to organize my thoughts.` | 过度外向的表述 |
| **功能驱动** | 每句解决一个自我表达的需求 | `I need some me-time to recharge.` | 功能模糊的句子 |
| **递进清晰** | 同一功能下 L1→L2→L3 难度阶梯明显 | `I like reading.`→`I prefer fiction over non-fiction.`→`I've been trying to read 20 pages a day.` | 全是同一难度 |
| **文化自然** | 符合英语国家谈论个人生活的习惯 | `I practice mindfulness to stay grounded.` | 生硬不自然的表达 |

### 适用对象

- **难度**：L1（零基础）→ L3（中级提高），同一场景内渐进
- **前置要求**：无（L1 起点）
- **目标学习者**：任何希望用英语表达个人内心世界、独处时光、兴趣爱好和自我成长的学习者

### 难度标识参考

| 难度 | 个人场景参考 | 语言特征 |
|:----:|------------|---------|
| 🧒 L1 | 简单喜好、日常活动 | 简单句、like/enjoy、基本动词 |
