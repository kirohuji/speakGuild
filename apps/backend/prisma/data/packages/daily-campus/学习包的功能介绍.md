# 校园功能场景 · 问候 · 吃饭 · 上课 · 考试 · 交友 · 课外 · 校园生活 · 求助办事 · 规划未来（L1~L3 递进）

## 学习包概览

本学习包是 **场景日常系列** 的核心组成部分，围绕 **9 大校园核心功能场景** 展开。在每个场景内，按 **L1→L2→L3** 的语言难度递进排列句块，让学习者清晰看到：**同一个交际需求，如何从最简单的表达一步步升级为更地道、更复杂的说法。**

> ❌ 传统做法：按年龄分"幼儿园英语→小学英语→中学英语→大学英语"，学习者只选自己当前年龄，看不到成长路径。
> ✅ 本包做法：按 **功能场景** 组织（如"吃饭"），展示 `More juice please` → `Want to try my lunch?` → `How much is the special?` → `Is there a vegetarian option?` 的完整进化链条。学习者可以从自己当前的难度入口切入，然后向上攀升。

### 为什么按功能场景递进？

| 传统按年龄分 | ✅ 本包按功能递进 |
|------------|----------------|
| 学习者只学自己年龄段，视野窄 | 同一功能下看到完整表达光谱 |
| 低段学过的内容高段不再出现 | 同一功能螺旋式上升，温故知新 |
| 脱离真实交际需求 | 每句都能回答"这个场景我到底该怎么说" |
| 不知道自己的水平该学什么 | 按 L1→L2→L3 清晰标识难度，对号入座 |

### 句块（chunk）vs 句型（pattern）

本包包含 **两类数据**，分工不同：

| | 句块（chunk） | 句型（pattern） |
|--|-------------|----------------|
| **是什么** | 可直接背诵的 **完整句子** | 带替换槽位的 **句型框架** |
| **作用** | 背下来直接用，培养语感 | 举一反三，替换关键词生成新句子 |
| **例子** | `Can I have some more juice, please?` | `Can I have some more ___, please?` |
| **对应文件** | `chunks.csv` | `sentence_patterns.csv` |

> **学习路径**：先按功能场景浏览句块→找到适合自己当前难度的句块背熟→用句型框架替换关键词举一反三→向上挑战更高难度的句块。

### 句块设计原则

| 原则 | 说明 | ✅ 好句块 | ❌ 差句块 |
|------|------|----------|----------|
| **功能驱动** | 每个场景解决一个真实校园交际需求 | 吃饭场景所有句块都跟"吃"有关 | 把问候混进吃饭场景 |
| **递进清晰** | 同一功能下 L1→L2→L3 难度阶梯明显 | `More juice`→`What's for lunch?`→`I'll have the chicken` | 全是同一难度 |
| **完整可背** | 完整句子，拿来就能用 | `Could you repeat that? I didn't catch it.` | `repeat`（孤立的词） |
| **文化恰当** | 符合英语国家的校园礼貌习惯 | `Would you mind if I joined your group?` | 语气生硬不礼貌 |

### 适用对象

- **难度**：L1（零基础）→ L3（中级提高），同一场景内渐进
- **前置要求**：无（L1 起点）
- **目标学习者**：任何需要用英语在学校场景中交流的学习者。不知道自己水平的可以从 L1 句块开始尝试，能轻松掌握就挑战 L2，以此类推

### 场景结构

每个场景按功能子主题划分，子主题内的句块按 **L1→L2→L3** 递进排列，并用"🧒→👦→🧑→👨‍🎓"示意该表达通常出现在哪个学段，帮助学习者建立年龄感。

| 难度标识 | 学校阶段参考 | 语言特征 |
|:--------:|------------|---------|
| 🧒 L1 | 幼儿园 | 单词句、祈使句、简单现在时 |
| 👦 L1 | 小学 | 简单疑问句、基本动词、there be |
| 🧑 L2 | 中学 | 复杂句、情态动词、过去时/将来时 |
| 👨‍🎓 L2~L3 | 大学 | 被动语态、虚拟语气、学术词汇 |
| 👨‍🎓 L3 | 大学高年级 | 复杂从句、正式语体、专业术语 |
