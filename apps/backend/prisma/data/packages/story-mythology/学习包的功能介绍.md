# 希腊神话 · 北欧神话

## 学习包概览

本学习包是 **故事系列** 的组成部分，通过东西方神话故事学习高阶英语表达。

### 适用对象

- **难度范围**：L3（中级提高）
- **前置要求**：掌握过去时和基本叙事能力
- **目标学习者**：对神话传说感兴趣、希望提升叙事能力的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 希腊神话·诸神与英雄 | 奥林匹斯众神、普罗米修斯、赫拉克勒斯 | L3 |
| 希腊神话·特洛伊与奥德赛 | 特洛伊战争、奥德修斯历险 | L3 |
| 北欧神话·诸神黄昏 | 奥丁索尔洛基、诸神黄昏与重生 | L3 |

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 3 个神话场景 |
| `training_topics.csv` | 7 个训练主题 |
| `scene_vocabulary.csv` | 基础词汇表 |
| `chunks.csv` | 核心叙事例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
