# 时间介词 · 时间连词与顺序

## 学习包概览

本学习包是 **语法专项系列** 的组成部分，系统学习英语中表达时间的各类词汇和结构。

### 适用对象

- **难度范围**：L2（中级基础）
- **前置要求**：掌握基本名词和简单句
- **目标学习者**：需要清晰表达时间关系的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 时间介词 | at/in/on + during/since/for/until/by | L2 |
| 时间连词与顺序 | before/after/first/then + when/while/as/until | L2 |

### 学习目标

完成本包学习后，学习者能够：

1. 区分 at(时间点)/in(时间段)/on(具体日期) 的用法
2. 使用 during/since/for/until/by 表达持续和起止
3. 用 before/after/first/then 描述事件的先后顺序
4. 用 when/while/as 表达同时发生的事件
5. 在口语和写作中准确表达时间关系
