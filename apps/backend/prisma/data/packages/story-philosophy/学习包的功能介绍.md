# 古希腊哲学 · 近现代哲学

## 学习包概览

本学习包是 **故事系列** 的组成部分，通过哲学思想史学习英语高阶表达和抽象思辨。

### 适用对象

- **难度范围**：L3（中级提高）
- **前置要求**：具备一定的抽象思维能力和词汇基础
- **目标学习者**：对哲学思想感兴趣、希望用英语讨论深刻话题的学习者

### 包含场景

| 场景名称 | 核心内容 | 难度 |
|---------|---------|:----:|
| 哲学·古希腊哲学 | 苏格拉底、柏拉图、亚里士多德 | L3 |
| 哲学·近现代哲学 | 笛卡尔、康德、尼采与存在主义 | L3 |

### 数据文件

| 文件 | 内容 |
|------|------|
| `scenes.csv` | 2 个哲学场景 |
| `training_topics.csv` | 6 个训练主题 |
| `scene_vocabulary.csv` | 基础词汇表 |
| `chunks.csv` | 核心表达例句 |
| `sentence_patterns.csv` | 句型模板 |
| `script_episodes.csv` | 学习剧集 |
| `episode_chunks.csv` | 剧集-例句关联 |
