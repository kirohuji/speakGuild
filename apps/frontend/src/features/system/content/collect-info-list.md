# 导游说个人信息收集清单

| 场景/业务功能 | 个人信息种类 | 目的 |
|--------------|-------------|------|
| 注册登录 | 手机号、验证码、密码、邮箱 | 用于注册创建账号和登录 |
| 注册登录 | 头像、昵称 | 用于完善网络身份标识 |
| 第三方登录 | 微信账号信息、Apple ID 信息 | 用于第三方账号授权登录 |
| 学习功能 | 练习记录、学习进度、收藏题目、生词本 | 用于完成核心备考学习功能 |
| 口语练习 | 语音录音、录音文本 | 用于 AI 发音评分和口语反馈（录音分析后即删除） |
| AI 评分 | 提交的题目内容、录音文本 | 发送至 DeepSeek API 进行评分分析 |
| TTS 语音 | 需要合成语音的文本内容 | 发送至 TTS 服务商生成标准发音 |
| 支付 | 支付时间、支付金额、支付渠道等支付信息 | 用于完成在线支付功能 |
| 客服与售后 | 用户与客服的联系记录、账号信息、身份核验信息、订单信息等 | 用于客服与售后服务及争议处理 |
| 附加功能 | 相机、相册、麦克风、通知 | 相机：扫码、拍摄照片、上传头像。相册：读取和保存图片、文件。麦克风：口语录音。通知：推送和事件提醒。上述附加功能仅在特定场景下用户使用相关功能时才会申请询问，用户拒绝不影响用户使用基本功能 |
| 使用过程中 | 设备信息、日志信息、网络信息、IP地址 | 用于安全保障和网络状态监控 |
| 系统功能 | 网络状态、存储空间、安装包信息 | 网络状态：监控网络变化。存储空间：保存应用数据、缓存、日志。安装包信息：用于应用更新 |
| 后台功能 | 远程通知、音频播放 | 支持应用在后台运行时的通知推送、音频播放等功能 |
