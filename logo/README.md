## 设计理念

Logo 以**对话气泡**为核心元素，体现"说"的品牌内涵；
气泡内融入**音波**元素，呼应"口语练习"的场景；
搭配**导向标识点**，暗合"导游"的职业属性。

整体采用**蓝青渐变**配色，传达专业、可信赖的品牌形象。
